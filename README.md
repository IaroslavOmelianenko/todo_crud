# Todo CRUD Application
A simple Todo application built with Spring Boot, PostgreSQL, RESTful API for managing tasks.

## Tech Stack
- Backend: Spring Boot, Spring Data JPA, Spring Web
- Database: PostgreSQL 
- Build Tool: Gradle 
- Java Version: 17
- Containerization: Docker, Docker Compose
- API Documentation: Swagger/OpenAPI

## Getting Started
1. Launch database
```
docker-compose up -d 
```
2. Launch app
```
./gradlew bootRun
```
